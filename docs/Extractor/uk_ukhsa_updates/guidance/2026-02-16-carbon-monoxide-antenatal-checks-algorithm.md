# Carbon monoxide (CO): antenatal checks algorithm for midwives

**發布日期：** 2026-02-16
**來源：** UK Health Security Agency (UKHSA)
**分類：** 指引文件

## 摘要

英國健康安全局（UKHSA）發布了助產士用於產前檢查的一氧化碳診斷演算法，旨在協助助產士識別孕婦一氧化碳中毒，並在暴露事件中提供諮詢建議。

## 背景

一氧化碳（CO）是一種無色、無味的有毒氣體，孕婦暴露於一氧化碳可能對母體和胎兒健康造成嚴重影響。此演算法為助產士提供系統化的檢查流程和臨床決策支援。

## 主要內容

### 目標對象
- 助產士
- 產前保健服務提供者

### 演算法功能
1. **診斷支援**：協助識別孕婦一氧化碳中毒的臨床徵象
2. **諮詢指引**：提供暴露後的處置建議
3. **風險評估**：系統化評估一氧化碳暴露風險

## 公共衛生意義

### 預防重點
- **產前篩檢**：將一氧化碳檢查納入常規產前評估
- **健康教育**：提升孕婦對一氧化碳風險的認知
- **早期介入**：及時發現並處理一氧化碳暴露

### 高風險情境
- 使用燃氣熱水器或暖氣設備
- 居住環境通風不良
- 使用木炭或燃煤取暖

## 臨床建議

### 助產士行動
1. 使用演算法進行系統化評估
2. 對高風險孕婦加強監測
3. 提供一氧化碳安全教育
4. 必要時轉診專科處理

### 孕婦注意事項
- 定期檢查燃氣設備
- 確保居住環境通風良好
- 了解一氧化碳中毒症狀（頭痛、暈眩、噁心）
- 安裝一氧化碳警報器
