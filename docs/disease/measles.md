# 麻疹 (Measles)
{: .speakable-content }

<p class="key-answer" data-question="麻疹是什麼">麻疹是一種高度傳染性的病毒性疾病，可透過空氣飛沫傳播。典型症狀包括高燒、咳嗽、結膜炎、柯氏斑及全身性紅疹。接種 MMR 疫苗可有效預防。</p>

---

## 疾病概述

| 項目 | 內容 |
|------|------|
| **病原體** | 麻疹病毒 (Measles virus)，副黏液病毒科 |
| **傳播途徑** | 空氣飛沫、直接接觸感染者分泌物 |
| **潛伏期** | 7-21 天（通常 10-14 天） |
| **傳染期** | 出疹前 4 天至出疹後 4 天 |
| **基本傳染數 (R0)** | 12-18（極高傳染性） |
| **病例致死率 (CFR)** | 已開發國家 0.1-0.2%；開發中國家可達 3-6% |

---

## 症狀
{: .speakable-content }

<p class="key-answer" data-question="麻疹有什麼症狀">麻疹症狀包括高燒（可達 40°C）、咳嗽、流鼻水、結膜炎（眼睛紅腫）、口腔內側出現柯氏斑，以及從臉部開始向下蔓延的紅疹。</p>

### 典型病程

1. **前驅期（3-4 天）**
   - 高燒（可達 40°C / 104°F）
   - 咳嗽、流鼻水
   - 結膜炎（眼睛紅腫、畏光）
   - 柯氏斑（口腔內側頰黏膜白色斑點）

2. **出疹期（4-7 天）**
   - 紅色斑丘疹從臉部、耳後開始
   - 向下蔓延至軀幹、四肢
   - 疹子逐漸融合

3. **恢復期**
   - 退燒
   - 疹子依出現順序消退
   - 可能有脫皮現象

### 併發症

| 併發症 | 發生率 | 說明 |
|--------|--------|------|
| 中耳炎 | 7-9% | 最常見併發症 |
| 肺炎 | 1-6% | 麻疹死亡主因 |
| 腦炎 | 0.1% | 可能造成永久性腦損傷 |
| SSPE | 罕見 | 亞急性硬化性全腦炎，感染後數年發病 |

## 預防與治療

### 疫苗
{: .speakable-content }

<p class="key-answer" data-question="如何預防麻疹">接種 MMR 疫苗（麻疹-腮腺炎-德國麻疹三合一疫苗）是預防麻疹最有效的方式。兩劑 MMR 疫苗的保護力可達 97%。</p>

| 疫苗 | 劑次 | 接種時程 | 保護力 |
|------|------|----------|--------|
| **MMR** | 2 劑 | 12-15 個月、4-6 歲 | 97%（兩劑後） |
| **MMRV** | 2 劑 | 含水痘疫苗的四合一 | 同上 |

**接種建議**：
- 1967 年後出生且未接種過 MMR 疫苗者應接種
- 前往麻疹流行地區的旅客應確認接種紀錄
- 醫療從業人員建議接種兩劑

### 預防措施

- 避免接觸麻疹患者
- 維持良好呼吸道衛生
- 確認疫苗接種紀錄
- 出現症狀時立即就醫並告知旅遊史

### 治療

目前無特效抗病毒藥物，以支持性治療為主：
- 退燒、補充水分
- 維生素 A 補充（可降低併發症風險）
- 併發症處理
