# 猴痘 (Mpox)
{: .speakable-content }

<p class="key-answer" data-question="猴痘是什麼">猴痘（Mpox，前稱 Monkeypox）是由猴痘病毒引起的病毒性疾病。主要透過密切接觸傳播，症狀包括發燒、淋巴結腫大及特徵性皮疹。目前 clade Ib 和 clade IIb 兩種變異株在全球流行。</p>

---

## 疾病概述

| 項目 | 內容 |
|------|------|
| **病原體** | 猴痘病毒 (Monkeypox virus)，正痘病毒屬 |
| **傳播途徑** | 密切接觸（皮膚、黏膜）、飛沫、污染物品 |
| **潛伏期** | 5-21 天（通常 6-13 天） |
| **傳染期** | 症狀出現至皮疹完全結痂脫落 |
| **病例致死率 (CFR)** | clade I: 1-10%；clade II: <1% |

### 病毒分支

| 分支 | 地理分布 | 嚴重度 | 備註 |
|------|----------|--------|------|
| **Clade I** | 中非 | 較高 | 病例致死率 1-10% |
| **Clade Ia** | 剛果民主共和國 | 較高 | 傳統地方性流行 |
| **Clade Ib** | 剛果、東非 | 中等 | 2024-2026 新興變異株 |
| **Clade II** | 西非 | 較低 | 病例致死率 <1% |
| **Clade IIb** | 全球 | 較低 | 2022-2026 全球流行株 |

## 症狀
{: .speakable-content }

<p class="key-answer" data-question="猴痘有什麼症狀">猴痘症狀包括發燒、頭痛、肌肉痠痛、淋巴結腫大，以及特徵性皮疹。皮疹通常從臉部開始，可出現在生殖器、肛門區域，經歷斑疹→丘疹→水泡→膿泡→結痂的演變過程。</p>

### 典型病程

1. **前驅期（0-5 天）**
   - 發燒
   - 劇烈頭痛
   - 淋巴結腫大（特徵性）
   - 肌肉痠痛、背痛
   - 疲倦

2. **出疹期（1-3 週）**
   - 皮疹演變：斑疹 → 丘疹 → 水泡 → 膿泡 → 結痂
   - 分布：臉部、手掌、腳掌、生殖器、肛門區域
   - 可伴隨疼痛或搔癢

3. **恢復期**
   - 結痂脫落（2-4 週）
   - 可能留下疤痕或色素變化

### 併發症

| 併發症 | 說明 |
|--------|------|
| 繼發性細菌感染 | 皮膚病灶感染 |
| 支氣管肺炎 | 呼吸道受累 |
| 腦炎 | 罕見但嚴重 |
| 角膜感染 | 可能影響視力 |
| 直腸炎 | 肛門區域病灶併發 |

## 預防與治療

### 疫苗
{: .speakable-content }

<p class="key-answer" data-question="如何預防猴痘">天花疫苗（如 JYNNEOS/Imvanex）可預防猴痘，保護效力約 85%。建議高風險族群（如暴露後預防）接種。避免與疑似病例密切接觸也是重要預防措施。</p>

| 疫苗 | 劑次 | 適用對象 | 保護力 |
|------|------|----------|--------|
| **JYNNEOS / Imvanex** | 2 劑 | 高風險族群、暴露後預防 | ~85% |
| **ACAM2000** | 1 劑 | 限制使用（副作用較多） | 較高 |

**接種建議**：
- 與確診者有密切接觸者（暴露後預防）
- 高風險族群（依各國指引）
- 醫療從業人員（照護猴痘患者）

### 預防措施

- 避免與疑似病例皮膚接觸
- 不共用毛巾、床單等個人物品
- 注意手部衛生
- 避免接觸野生動物（尤其齧齒類）

### 治療

- **支持性治療**為主
- **Tecovirimat (TPOXX)**：抗病毒藥物，用於嚴重病例
- 疼痛控制
- 預防繼發性細菌感染
