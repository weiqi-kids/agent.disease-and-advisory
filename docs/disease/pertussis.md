# 百日咳 (Pertussis)
{: .speakable-content }

<p class="key-answer" data-question="百日咳是什麼">百日咳是由百日咳桿菌引起的高度傳染性呼吸道疾病。特徵是劇烈陣發性咳嗽，伴隨吸氣時的「雞啼聲」。對嬰幼兒特別危險，可透過 Tdap 疫苗預防。</p>

---

## 疾病概述

| 項目 | 內容 |
|------|------|
| **病原體** | 百日咳桿菌 (Bordetella pertussis) |
| **傳播途徑** | 飛沫傳播、直接接觸感染者分泌物 |
| **潛伏期** | 7-10 天（範圍 4-21 天） |
| **傳染期** | 症狀出現至治療後 5 天或咳嗽後 3 週 |
| **基本傳染數 (R0)** | 12-17（極高傳染性） |
| **病例致死率 (CFR)** | 嬰兒可達 1-2%；成人罕見死亡 |

---

## 症狀
{: .speakable-content }

<p class="key-answer" data-question="百日咳有什麼症狀">百日咳典型症狀為劇烈陣發性咳嗽，可持續數週甚至數月。咳嗽後常伴隨吸氣時的「雞啼聲」（whoop），以及咳後嘔吐。嬰兒可能出現呼吸暫停。</p>

### 典型病程

1. **卡他期（1-2 週）**
   - 類似感冒症狀
   - 輕微咳嗽、流鼻水
   - 低燒或無發燒
   - **此期傳染力最強**

2. **陣發期（2-8 週）**
   - 劇烈陣發性咳嗽
   - 吸氣時「雞啼聲」
   - 咳後嘔吐
   - 臉部發紺（缺氧）
   - 嬰兒可能呼吸暫停

3. **恢復期（數週至數月）**
   - 咳嗽逐漸減輕
   - 可能持續數月
   - 呼吸道感染可使症狀復發

### 高風險族群

| 族群 | 風險 | 說明 |
|------|------|------|
| 未滿 1 歲嬰兒 | 🔴 極高 | 併發症及死亡風險最高 |
| 孕婦 | 🟡 中 | 可傳給新生兒 |
| 未完整接種者 | 🟡 中 | 保護力不足 |
| 醫療從業人員 | 🟡 中 | 暴露風險高 |

### 併發症

| 併發症 | 好發族群 | 說明 |
|--------|----------|------|
| 肺炎 | 嬰兒 | 最常見嚴重併發症 |
| 腦病變 | 嬰兒 | 缺氧導致 |
| 呼吸暫停 | 嬰兒 | 可能致命 |
| 肋骨骨折 | 成人 | 劇烈咳嗽導致 |

## 預防與治療

### 疫苗
{: .speakable-content }

<p class="key-answer" data-question="如何預防百日咳">接種含百日咳成分的疫苗（DTaP 或 Tdap）是預防百日咳最有效的方式。建議孕婦每次懷孕都接種 Tdap 疫苗，將抗體傳給新生兒。</p>

| 疫苗 | 適用對象 | 劑次 | 說明 |
|------|----------|------|------|
| **DTaP** | 嬰幼兒 | 5 劑 | 2、4、6、15-18 個月及 4-6 歲 |
| **Tdap** | 青少年/成人 | 1 劑 + 追加 | 11-12 歲，每 10 年追加 |
| **Tdap（孕婦）** | 孕婦 | 每次懷孕 | 27-36 週接種，保護新生兒 |

**接種建議**：
- 孕婦每次懷孕第三孕期應接種 Tdap
- 嬰兒照護者應確認 Tdap 接種
- 醫療從業人員應定期追加

### 預防措施

- 確認疫苗接種紀錄
- 避免接觸疑似病例
- 咳嗽禮儀（遮口鼻）
- 勤洗手

### 治療

- **抗生素**：Azithromycin、Clarithromycin 或 Erythromycin
- 早期治療可縮短傳染期
- 支持性治療（氧氣、補液）
- 嬰兒可能需住院觀察
