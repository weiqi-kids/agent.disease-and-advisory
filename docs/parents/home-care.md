# 兒童居家照護指南
{: .speakable-content }

<p class="key-answer" data-question="孩子生病如何居家照護">居家照護重點包括：監測體溫、補充水分、充分休息、維持舒適環境。觀察警示徵象，必要時就醫。</p>

孩子生病時的居家照護要點與就醫時機判斷。

---

## 發燒處理

### 何謂發燒？

| 測量部位 | 發燒標準 |
|----------|----------|
| 肛溫/耳溫 | ≥ 38°C |
| 口溫 | ≥ 37.5°C |
| 腋溫 | ≥ 37.2°C |

### 居家處置

1. **監測體溫**：每 4-6 小時測量
2. **補充水分**：少量多次給水、電解質液
3. **物理降溫**：溫水擦澡、減少衣物
4. **藥物退燒**：依體重給予適當劑量

<p class="key-answer" data-question="孩子發燒要立即退燒嗎">輕微發燒是身體對抗感染的正常反應，不必急於完全退燒。當孩子不舒服或體溫過高時再給予退燒藥。</p>

### ⚠️ 需立即就醫

- 3 個月以下嬰兒發燒
- 體溫超過 40°C
- 發燒超過 3 天
- 伴隨抽搐
- 精神萎靡、嗜睡
- 呼吸困難
- 頸部僵硬

---

## 腹瀉處理

### 居家處置

1. **補充水分**：口服電解質液（如寶礦力稀釋、ORS）
2. **清淡飲食**：白稀飯、白吐司、香蕉
3. **避免刺激**：暫停乳製品、油膩食物
4. **觀察排便**：記錄次數、性狀

### 電解質補充建議

| 年齡 | 每次腹瀉後補充量 |
|------|------------------|
| < 2 歲 | 50-100 ml |
| 2-10 歲 | 100-200 ml |
| > 10 歲 | 依口渴程度 |

### ⚠️ 需立即就醫

- 血便或黏液便
- 嚴重腹痛
- 高燒不退
- 脫水徵象（尿量減少、口乾、哭無淚）
- 6 小時以上無尿（嬰幼兒）
- 腹瀉超過 3 天

---

## 咳嗽處理

### 居家處置

1. **保持濕潤**：使用加濕器、多喝溫水
2. **抬高頭部**：睡眠時稍微墊高
3. **避免刺激**：遠離煙霧、灰塵
4. **蜂蜜水**：1 歲以上可給蜂蜜水緩解（1 歲以下禁止）

### ⚠️ 需立即就醫

- 呼吸急促或困難
- 吸氣時有喘鳴聲
- 嘴唇發紫
- 咳血
- 咳嗽超過 2 週
- 伴隨高燒

---

## 嘔吐處理

### 居家處置

1. **暫停進食**：嘔吐後 1-2 小時暫不進食
2. **少量多次**：從小口水開始，逐漸增加
3. **避免脫水**：給予電解質液
4. **觀察情況**：記錄嘔吐次數

### ⚠️ 需立即就醫

- 劇烈頭痛後嘔吐
- 嘔吐物含血或咖啡色物質
- 腹部劇痛
- 6 小時以上無法進食
- 脫水徵象

---

## 脫水警示

學會辨識脫水徵象：

| 徵象 | 輕度脫水 | 中重度脫水（需就醫） |
|------|----------|----------------------|
| 口腔 | 稍乾 | 非常乾燥 |
| 眼眶 | 正常 | 凹陷 |
| 眼淚 | 有 | 哭不出眼淚 |
| 尿量 | 稍減少 | 明顯減少/無尿 |
| 皮膚 | 正常 | 乾燥、彈性差 |
| 精神 | 略煩躁 | 嗜睡或躁動 |

## 居家常備藥物

| 藥物 | 用途 | 注意事項 |
|------|------|----------|
| 退燒藥（普拿疼/依普芬） | 退燒、止痛 | 依體重計算劑量 |
| 口服電解質液 | 補充電解質 | 腹瀉嘔吐時使用 |
| 生理食鹽水 | 清潔鼻腔 | 鼻塞時使用 |
| OK 繃、消毒藥水 | 外傷處理 | |
| 體溫計 | 測量體溫 | 電子式較安全 |

## 就醫判斷流程

```
孩子不舒服
    │
    ├── 出現警示徵象？ ──是──→ 立即就醫/急診
    │
    └── 否
         │
         ├── 症狀輕微，精神活動力好 → 居家觀察
         │
         └── 症狀持續或惡化 → 門診就醫
```

## 常見問題

### 發燒可以洗澡嗎？

可以。溫水澡有助散熱和舒適，但避免水溫過冷或過熱。洗完後擦乾、穿適量衣物。

### 生病時可以吃冰淇淋嗎？

喉嚨痛時冰涼食物可能緩解不適，但腸胃炎時建議避免。

### 需要吃抗生素嗎？

抗生素只對細菌感染有效，大部分感冒是病毒引起，不需要抗生素。請遵循醫師指示。
