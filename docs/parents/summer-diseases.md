# 夏季傳染病預防
{: .speakable-content }

<p class="key-answer" data-question="夏季要注意哪些傳染病">夏季常見傳染病包括登革熱、腸病毒、細菌性腸胃炎（沙門氏菌、大腸桿菌）。防蚊、勤洗手、注意飲食衛生是主要預防措施。</p>

夏季常見傳染病的認識與預防方法。

---

## 登革熱

### 認識登革熱

| 項目 | 說明 |
|------|------|
| 傳染途徑 | 埃及斑蚊、白線斑蚊叮咬 |
| 潛伏期 | 3-14 天（通常 4-7 天） |
| 流行季節 | 5-11 月（夏秋季） |
| 流行地區 | 台灣南部、東南亞、中南美洲 |

### 症狀

<p class="key-answer" data-question="登革熱有什麼症狀">登革熱典型症狀包括突發高燒、劇烈頭痛、眼窩痛、肌肉關節痛、出疹。兒童症狀可能較不典型。</p>

- 🔥 突發高燒（39-40°C）
- 😣 劇烈頭痛、眼窩痛
- 💪 肌肉、骨頭、關節疼痛（「斷骨熱」）
- 🔴 出疹（發病後 3-4 天）
- 🩸 出血傾向（牙齦出血、瘀青）

### ⚠️ 警示徵象（需立即就醫）

- 腹部劇痛
- 持續嘔吐
- 牙齦出血、流鼻血
- 皮下出血（瘀青）
- 嗜睡、躁動
- 尿量明顯減少

### 預防方法

**清除孳生源（4 訣）**：
1. 「巡」— 巡視居家環境，清除積水容器
2. 「倒」— 倒掉積水
3. 「清」— 清洗容器內壁
4. 「刷」— 刷洗去除蟲卵

**個人防護**：
- 穿淺色長袖長褲
- 使用防蚊液（含 DEET 或派卡瑞丁）
- 安裝紗窗紗門
- 使用蚊帳

## 腸病毒

### 認識腸病毒

| 項目 | 說明 |
|------|------|
| 傳染途徑 | 糞口傳染、飛沫傳染、接觸傳染 |
| 潛伏期 | 3-5 天 |
| 好發年齡 | 5 歲以下幼童 |
| 流行季節 | 4-9 月（春夏季） |

### 症狀

- 🔥 發燒
- 👄 口腔潰瘍（疱疹性咽峽炎）
- 🖐️ 手足口病（手、腳、口腔水疱）
- 😢 食慾不振、流口水

### ⚠️ 重症前兆（需立即就醫）

- 嗜睡、意識不清
- 肌躍型抽搐（突然全身肌肉收縮）
- 持續嘔吐
- 呼吸急促或心跳加快
- 手腳無力

<p class="key-answer" data-question="腸病毒重症前兆有哪些">腸病毒重症前兆包括嗜睡、肌躍型抽搐、持續嘔吐、呼吸急促、手腳無力，出現這些徵象需立即就醫。</p>

### 預防方法

- ✅ 勤洗手（濕搓沖捧擦）
- ✅ 不共用餐具、毛巾
- ✅ 玩具定期清潔消毒
- ✅ 保持居家通風
- ✅ 流行期避免出入人潮擁擠場所

### 居家照護

- 給予冰涼、軟質食物（布丁、冰淇淋、優格）
- 避免酸、辣、熱的食物
- 補充水分
- 口腔潰瘍可用醫師開立的止痛藥

## 細菌性腸胃炎

### 常見病原

| 病原 | 常見來源 | 潛伏期 |
|------|----------|--------|
| 沙門氏菌 | 蛋類、肉類 | 6-72 小時 |
| 金黃色葡萄球菌 | 奶製品、肉類 | 1-7 小時 |
| 大腸桿菌 | 生菜、生肉 | 1-10 天 |
| 腸炎弧菌 | 海鮮 | 4-96 小時 |

### 症狀

- 🤮 噁心、嘔吐
- 💩 腹瀉（可能帶血或黏液）
- 🔥 發燒
- 😣 腹部絞痛

### 預防方法

**食物處理五要原則**：

| 要點 | 說明 |
|------|------|
| 要洗手 | 處理食物前後洗手 |
| 要新鮮 | 選擇新鮮食材 |
| 要生熟食分開 | 砧板、刀具分開 |
| 要徹底加熱 | 食物中心溫度達 70°C |
| 要注意保存溫度 | 熱食 >60°C，冷食 <5°C |

### 居家照護

- 暫時禁食 2-4 小時讓腸胃休息
- 少量多次補充水分和電解質
- 清淡飲食（白稀飯、白吐司）
- 避免乳製品、油膩食物

## 夏季防疫總整理

| 疾病 | 主要預防 | 關鍵提醒 |
|------|----------|----------|
| 登革熱 | 防蚊、清除積水 | 出國東南亞要注意 |
| 腸病毒 | 勤洗手 | 5 歲以下注意重症 |
| 腸胃炎 | 飲食衛生 | 生熟食分開 |
| 日本腦炎 | 接種疫苗、防蚊 | 15 個月起接種 |

## 常見問題

### 登革熱可以吃退燒藥嗎？

可以，但**避免使用阿斯匹靈**，因為可能增加出血風險。建議使用普拿疼（乙醯胺酚）。

### 得過一次登革熱會有免疫力嗎？

只對同一型別有免疫力。登革熱有 4 種型別，感染不同型別時可能症狀更嚴重。

### 腸病毒一定會得手足口病嗎？

不一定。腸病毒有多種型別，可能只有發燒、口腔潰瘍（疱疹性咽峽炎），不一定出現手足水疱。
