### 1.2 排除範圍

以下項目不在本次改版範圍：

| 項目 | 排除原因 |
|------|----------|
| **原生行動 App** | 網站已響應式設計，短期無需開發 App，長期可考慮 PWA（列入 Phase 3） |
| **互動式全球地圖** | 技術複雜度高，與核心定位（文字情報）不符，可以簡易地圖替代（Phase 3） |
| **即時更新（15 分鐘級）** | 定位為週報式整合分析，無需即時更新，現有更新頻率已足夠 |
| **臨床診斷支援** | 超出定位範圍，需醫療專業知識與法律合規，風險高 |
| **Crowdsourcing 參與式監測** | 專注官方資料，民眾回報數據可靠性低 |
| **付費訂閱功能** | 定位為免費開放平台，與品牌價值不符 |
| **多語言（除英文外）** | 資源有限，優先提供英文版（Phase 2），其他語言暫不考慮 |
| **社群論壇/討論區** | 維運成本高，短期建議使用外部社群（Discord/Telegram），列入 Phase 3 評估 |

```
1. Hero 區塊
   - 內容：網站名稱、一句話描述、主要 CTA

   - 內容：表格（疾病、地區、嚴重度🔴🟡🟢、摘要、連結）

   - 內容：本週整合 X 個官方公告、涵蓋 Y 種疾病、Z 個國家
   - 目的：視覺化展示資料量與價值

4. Logo 牆（新增）
   - 內容：WHO、US CDC、ECDC、UKHSA、Taiwan CDC 的 Logo + 連結

```


**旅遊警示**：
```
- 高風險地區：建議暫緩非必要旅遊（Rwanda - Marburg 病毒）

- 低風險地區：一般預防即可
```
