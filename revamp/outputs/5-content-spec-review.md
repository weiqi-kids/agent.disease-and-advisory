## 問題清單

### 必須修改 (Blockers)

**無**

### 建議改善 (Suggestions)

1. **Logo 圖檔準備**：執行前需確認各官方機構 Logo 使用授權
   → 建議：優先使用官方提供的可外連 Logo 或文字替代

2. **地理篩選功能**：JavaScript 互動功能規格待補充
   → 建議：Phase 1 先用 CSS 類別標記，Phase 2 加入 JS 互動

3. **Schema JSON-LD**：首頁有 WebSite Schema，但缺少 FAQPage
   → 建議：執行時在「關於區塊」加入 FAQPage Schema
